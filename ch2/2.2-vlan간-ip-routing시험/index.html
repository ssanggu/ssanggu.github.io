2.2 VLAN간 ip routing시험

[구성도]

LAB구성파일: 20161222_L3-VLAN.zip

[LAB의 목적]

L3를 수행하는 백본스위치 1대를 놓고 서로 다른 VLAN간에 ip routing을 수행하여 서로 다른 VLAN간에 통신 시험

[습득해야할 내용]

  • Cisco IOL에서 스위치 이미지를 사용할때
    • L3 라우팅기능을 사용하려면 ip routing
    • L3 스위치에서 default gateway를 지정하려면 ip route 0.0.0.0 0.0.0.0 nexthop으로 지정
    • L2스위치로만 사용하려면 no ip routing
    • L2 스위치에서 default gateway를 지정하려면 ip default-gateway nexthop으로 지정
