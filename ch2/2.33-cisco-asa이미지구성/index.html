[LAB의 목적]

Cisco ASA이미지를 UnetLab에 구성하여 LAB상에서 방화벽장비로 이용하기 위한 LAB입니다. ASA이미지를 구성한후에 기본적인 설정과 ASDM을 통해서 ASA방화벽을 연결한후 기본적인 사용법을 익혀보세요.

[LAB수행조건]

  • Cisco ASA이미지를 unetLab에 환경구성하기
  • 모든 노드가 통신이 가능하도록 환경구성하기
  • ASA의 방화벽정책을 ALL Permit정책으로 설정하기
  • ASA 관리포트 연결이 호스트OS에서 가능하도록 구성하기
    • http server enable
    • http 접근ip허용
  • 호스트OS에서 ASA관리 연결 및 기본 사용법 숙지하기
